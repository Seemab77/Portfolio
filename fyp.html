Final Year Design Project

Smart Healthcare IoT — Diabetes Management

A comprehensive non-invasive hardware-software pipeline designed for continuous light-spectroscopy glucose monitoring and real-time ML-based telemetry forecasting.

🚀 Capstone Focus

This project eliminates painful daily finger-pricks by deploying **Near-Infrared (NIR) spectroscopy** to measure blood-glucose concentration through dermal light-absorption parameters. The sensor node processes raw inputs via adaptive signal filters and transmits telemetry to a custom neural network for inference.

92% ML Correlation (R²)
<15ms Inference Latency
100% Non-Invasive

Key Project Outcomes

  • 1 Designed and calibrated an analog sensor circuit utilizing specific wavelengths (940nm-1050nm) to measure dermal blood-glucose absorption rates.
  • 2 Preprocessed absorption signals through discrete filtering to remove movement artifacts and pulse pressure fluctuations.
  • 3 Trained a TensorFlow-based Multi-Layer Perceptron (MLP) model, achieving 92% correlation (R²) with clinical capillary blood glucose meters.
  • 4 Programmed an ESP32 in Embedded C++ to handle sensor ADC sampling, baseline calibration, and Bluetooth Low Energy (BLE) transmissions.
  • 5 Created a cross-platform mobile application providing real-time data streaming, trend forecasting, and high/low threshold notifications.

// ESP32 IR Spectroscopy Dermal Absorption Sampling
#include <Arduino.h>
#define NIR_LED_PIN 14
#define ADC_INPUT_PIN 34
#define NUM_SAMPLES 128

float sampleNIR() {
    float sum = 0.0;
    digitalWrite(NIR_LED_PIN, HIGH);
    delayMicroseconds(50); // Stabilization window
    
    for (int i = 0; i < NUM_SAMPLES; i++) {
        sum += analogReadMilliVolts(ADC_INPUT_PIN);
        delayMicroseconds(10);
    }
    digitalWrite(NIR_LED_PIN, LOW);
    
    // Compute mean voltage absorption index
    return sum / NUM_SAMPLES;
}
# TensorFlow MLP Dermal Glucose Prediction Pipeline
import tensorflow as tf
from tensorflow.keras import layers, models

def build_mlp_predictor(input_shape=(8,)):
    model = models.Sequential([
        layers.Input(shape=input_shape),
        layers.Dense(64, activation='relu'),
        layers.Dropout(0.1),
        layers.Dense(32, activation='relu'),
        layers.Dense(1, activation='linear') # Output: Blood Glucose
    ])
    
    model.compile(optimizer='adam', loss='mse', metrics=['mae'])
    return model
